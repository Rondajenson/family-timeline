To add events: Edit the events array in the JavaScript section below.

  • startDate: Event date (YYYY-MM-DD format)
  • endDate: End date (use startDate for single events)
  • title: Event name
  • category: Life Line, Wedding, Birthday, Cars, or Houses
  • description: Optional details
  • image: URL to photo (or relative path in images/)

Notes: All events display in a single timeline. Use the filter buttons to show specific categories or "All" to show everything.
